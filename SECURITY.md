# Security Policy

This repository is a synthetic, public-safe SOC detection lab. It is not a
production system, managed service, or deployable security product.

The public package should not contain real credentials, secret values, tenant
identifiers, private user data, production screenshots, or private preparation
material.

If you notice material that appears sensitive or unsafe to publish, please
contact the repository owner privately. Do not open a public issue that repeats
the sensitive material.

Before future public updates, run `tools/verify-public-safe.ps1` from the
repository root and review the rendered dashboard, documentation, and deck for
disclosure issues.
